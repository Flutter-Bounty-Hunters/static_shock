Welcome to your Static Shock website. You've chosen the empty template, which includes the minimum set of files to build and serve a website.

To get started:

  1. Open this index.html file and start adjusting for your content needs.
    • Consider switching this file from .html to .jinja to take advantage of template variables.
  2. Add all images (e.g., in /source/images), styles (e.g., in /source/styles), and JavaScript to the project that you need.
  3. Open the Dart file under /bin and configure Static Shock to pick any files you want, and add any plugins you need.
